\section{Introduction and Background}

Flow Matching \citep{lipman2023flow} trains continuous normalizing flows by regressing a neural velocity field $\vt(x,t)$ against a target vector field that transports a prior $p_0 = \mathcal{N}(0,I)$ to data $p_1$. Conditional Flow Matching (CFM) makes this tractable by decomposing the marginal target into conditional terms. For each data point $x_1$, a conditional probability path $p_t(x|x_1)$ defines a Gaussian interpolation, and the CFM loss regresses against the conditional vector field:
\begin{equation}
    \mathcal{L}_{\text{CFM}}(\theta) = \E_{t,x_0,x_1} \left\| \vt(\xt, t) - \ut(\xt | x_1) \right\|^2.
    \label{eq:cfm_loss}
\end{equation}
Under the \textbf{OT probability path} \citep{lipman2023flow}, the interpolation is linear: $\mu_t(x_1) = t \cdot x_1$, $\sigma_t = 1-(1-\sigma_{\min})t$ with $\sigma_{\min}=10^{-4}$, yielding:
\begin{equation}
    \ut(x|x_1) = \frac{x_1 - (1-\sigma_{\min})x}{1-(1-\sigma_{\min})t}.
    \label{eq:ot_vf}
\end{equation}

While unbiased, the single-sample conditional target has high variance. Three techniques address this along independent axes:

\paragraph{BatchOT coupling.} \citet{tong2024improving} replace independent noise--data pairing with minibatch optimal transport: within each batch, Sinkhorn's algorithm \citep{cuturi2013sinkhorn} finds a low-cost assignment between noise and data, producing shorter, straighter interpolation paths.

\paragraph{StableVM objective.} For each $\xt$, $K{=}16$ references $\{x_1^{(k)}\}$ are drawn from a memory bank. Importance weights $w_k = \softmax_k(\log p_t(\xt|x_1^{(k)}))$ yield a multi-reference target $\ut^{\text{VM}} = \sum_k w_k \ut(\xt|x_1^{(k)})$. The log-probability under the OT path is:
\begin{equation}
    \log p_t(x|x_1) = -\tfrac{d}{2}\log(2\pi) - d\log\sigma_t - \tfrac{1}{2\sigma_t^2}\|x - \mu_t(x_1)\|^2.
    \label{eq:log_prob}
\end{equation}

\paragraph{TPC estimator.} Antithetic pairs $t_1 \sim \mathcal{U}[0,0.5]$, $t_2=1-t_1$ replace uniform sampling, with a consistency penalty:
\begin{equation}
    \mathcal{L}_{\text{TPC}} = \mathcal{L}(t_1) + \mathcal{L}(t_2) + \lambda\|\vt(\xt^{(1)},t_1) - \vt(\xt^{(2)},t_2)\|^2, \quad \lambda=0.1.
    \label{eq:tpc}
\end{equation}

\paragraph{Research questions.} Each technique works in isolation, but their composition is unexplored. We ask: \textbf{(RQ1)}~Are the three axes additive? \textbf{(RQ2)}~Which single axis helps most? \textbf{(RQ3)}~Do 2D geometric insights transfer to CIFAR-10?
